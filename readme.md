## 🌀 Overview

**Identity Vortex** is a stunning, production-grade web application that generates complete fake identities (personas) with every refresh. Each click produces:

- ✅ Realistic **full name** (gender-appropriate)
- ✅ **Street address** with city, state & ZIP
- ✅ Fake **Social Security Number** (SSN)
- ✅ Fake **Credit Card Number** + Expiry + CVV
- ✅ **Realistic person photo** matching the generated gender
- ✅ All data is **100% synthetic** — no real PII, completely safe

Perfect for:
- UI/UX design mockups
- Testing forms & validation
- Roleplaying & TTRPG characters
- Portfolio showcase projects
- Privacy-safe demo data
